# 📊 HR Analytics Dashboard

A comprehensive Human Resources analytics project built using **Microsoft Excel**, **Power BI**, and **DAX** to uncover workforce trends, compensation patterns, and performance insights.

---

## 🔍 Project Overview

This dashboard transforms raw HR data into interactive visuals that help HR managers and business leaders make data-driven decisions around employee management, compensation benchmarking, and departmental planning.

---

## 📁 Files in this Repository

| File | Description |
|------|-------------|
| `HR Analytics.pbix` | Power BI report file |
| `HR_Data.xlsx` | Source dataset (Excel) |

---

## 📌 Key Metrics & Insights

- 👥 **Total Employees:** 1,470
- ♀️ **Gender Split:** 60% Female | 40% Male
- 🏢 **Top Department:** Research & Development (961 employees)
- 💰 **Highest Monthly Rate Role:** Sales Executive (4.7M+)
- ⭐ **Avg Performance Rating:** ~3.1 – 3.2 across all roles

---

## 📈 Dashboard Visuals

- Monthly Rate by Job Role (bar chart)
- Daily Rate by Job Role (bar chart)
- Hourly Rate by Job Role (bar chart)
- Average Rating by Job Role (dot/line chart)
- Gender Count (pie chart)
- Department Count (bar chart)
- Job Role Count (horizontal bar chart)

## 🛠️ Tools & Technologies

| Tool | Usage |
|------|-------|

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop** (free download from Microsoft)
3. Explore the interactive visuals using slicers and filters
